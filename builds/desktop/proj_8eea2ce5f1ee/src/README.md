# Liste de Courses

Cette application permet de gérer votre liste de courses avec la possibilité d'ajouter, de supprimer et de cocher des articles. Elle utilise React pour l'interface, TailwindCSS pour le style, et elle est conçue pour être utilisée hors ligne grâce à un Service Worker.

## Installation

1. Téléchargez ou clonez ce dépôt.
2. Ouvrez `index.html` dans votre navigateur pour utiliser l'application.

## Fonctionnalités

- Ajouter des articles à la liste de courses.
- Supprimer des articles de la liste.
- Cocher des articles comme achetés.
- Stockage des données dans le navigateur pour y accéder ultérieurement.

## Technologies utilisées

- React 18
- TailwindCSS
- Service Worker

## Aide

Pour toute question, veuillez créer une issue sur le dépôt.
